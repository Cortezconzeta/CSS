Selector de clase

El selector de clase es de tipo, por eso, varios elementos pueden pertenecer a la misma clase. Se establecen:

  • En HTML, mediante el atributo class, que permite crear una clase dentro de un elemento. class=”nombre_de_clase” FLECHA Ej.: class=”estilo_1”
  • En CSS, mediante el punto. .nombre_de_clase FLECHA Ej.: .estilo_1

Para hacerlo con VSC, escribimos: nombre_etiqueta.nombre_de_clase
